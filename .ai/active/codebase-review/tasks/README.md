# Codebase Review Tasks

This directory holds remediation task plans that sit downstream of the codebase review findings.

Use this for execution-oriented plans tied to specific findings, for example:

- `a-003-react-query-migration.md`

Keep canonical review artifacts in the parent directory. Use task files here when a finding needs a larger implementation plan than fits comfortably in `architecture-findings.md` or `findings.md`.
